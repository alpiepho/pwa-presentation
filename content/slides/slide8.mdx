## simple-score-volleyball-pwa

- scoring [app](https://github.com/alpiepho/simple-score-volleyball-pwa) for volleyball
- hosted on Netlify
- uses Netlify Function (ie AWS Lambda) to access Twillio
- uses Twillio to send texts
- originally slated to be React Native
